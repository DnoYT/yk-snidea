# YK Snidea - 网页版 AI 对话开发者的福音

**YK Snidea** 是一款由 **广西云科互动科技有限公司** 开发的 VS Code 插件。它的诞生是为了解决当今 AI 辅助编程生态中的一个痛点：在面对高额的 API Token 翻译计费、严格的地区网络限制，以及强绑定的特定开发软件（如 Cursor）时，如何低成本且高效地利用顶尖大模型？

本插件通过精准提取本地代码和数据库上下文，为你一键拼装高质量的提示词（Prompt），让你直接在网页端（如 ChatGPT, Claude, DeepSeek）即可获得媲美专业 AI IDE 的深度开发体验。

## 🌟 核心理念

相比于动辄消耗海量且昂贵 API 额度的全自动开发模式，许多开发者更倾向于购买实惠的网页端 Pro 会员。然而，网页端大模型天然缺乏对本地工作区代码和数据库结构的“全局感知”。

**YK Snidea** 完美填补了这一空白：它作为一个高效的**本地上下文拼装器**，能将复杂的项目背景转化为 AI 能够理解的结构化数据。虽然它不包揽全自动代码写入，但它能让你以极低的成本，通过“复制粘贴”的方式，让高阶模型完全理解你的本地开发环境。

## ✨ 主要功能

* **🎭 角色管理**：预设不同的 AI 角色（System Prompt），如“资深全栈”、“SQL 优化专家”，确保 AI 的回复专业对口。
* **🔗 深度上下文关联**：
  * 使用 `@`：快速搜索并读取工作区文件源码。
  * 使用 `#`：自动读取 MySQL 数据库表结构（DDL）。
* **📑 开发规范绑定**：将公司或项目的开发规范（Profile）一键注入，让 AI 写的代码自带“大厂风格”。
* **💾 智能提示词存档**：生成的 Prompt 支持一键保存为 `.ykide/` 下的 Markdown 文件，方便回溯和二次迭代。

## 🛠️ 使用指南

1. **初始化**：在侧边栏点击图标打开 **YK Snidea**。
2. **连数据库**：配置 MySQL 连接，确保插件能读取你的表结构。
3. **写需求**：在输入框通过 `@` 引用关联代码，用 `#` 引用相关表名。
4. **拿结果**：点击“生成提示词”，一键复制到网页端大模型，开启你的高效对话。

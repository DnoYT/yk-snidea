# YK Snidea - 网页版 AI 对话开发者的福音

**YK Snidea** 是一款由 **广西云科互动科技有限公司** 开发的 VS Code 插件。它的诞生是为了解决当今 AI 辅助编程生态中的一个痛点：在面对高额的 API Token 翻译计费、严格的地区网络限制，以及强绑定的特定开发软件（如 Cursor）时，如何低成本且高效地利用顶尖大模型？

本插件通过精准提取本地代码和数据库上下文，为你一键拼装高质量的提示词（Prompt），让你直接在网页端（如 ChatGPT, Claude, DeepSeek）即可获得媲美专业 AI IDE 的深度开发体验。

## 🌟 核心理念

相比于动辄消耗海量且昂贵 API 额度的全自动开发模式，许多开发者更倾向于购买实惠的网页端 Pro 会员。然而，网页端大模型天然缺乏对本地工作区代码和数据库结构的“全局感知”。

**YK Snidea** 完美填补了这一空白：它作为一个高效的**本地上下文拼装器**，能将复杂的项目背景转化为 AI 能够理解的结构化数据。我们严格遵循 **Early Return (无 else)** 代码规范，确保插件运行的高性能与逻辑清晰度。

## ✨ 主要功能

* **🎭 角色预览化管理**：支持自定义 AI 角色（System Prompt），主界面选择角色后可立即预览角色设定的详细内容。
* **🔗 深度上下文关联**：
  * **使用 `@`**：模糊搜索并读取源码。支持**同名文件自动路径区分**，排序算法自动将隐藏文件（.开头）置后。
  * **使用 `#`**：自动检索 MySQL 表结构。悬浮可预览表 DDL 信息，自动注入 SQL 结构。
  * **🌳 结构感知**：支持一键生成项目文件树。通过工作区 `.ykide/.yktree` 配置过滤规则（语法同 .gitignore），精准控制上下文长度。
* **🚀 自动化代码应用 (Diff/Replace)**：
  * **智能解析**：支持解析 AI 返回的 `SEARCH/REPLACE` 格式代码块。
  * **无缝替换**：点击“应用剪贴板 Diff”即可自动执行代码替换，支持**自动创建目录**及**新建文件**，告别繁琐的手动复制。
* **🌳 模块化开发规范 (Profile)**：
  * **原子规则池**：支持按技术栈前缀（如 `[vue]`, `[php]`）对规则进行分组管理。
  * **树形交互**：采用 Element Plus 树形控件，支持分类展开收起、一键全选/反选/取消全选。
  * **动态微调**：在生成前，可基于合集快速手动增减特定的原子规则，灵活应对不同任务。
* **💾 数据备份与存档**：
  * **配置导出**：一键将全局 Role/Rule/Profile 配置导出至工作区 `.ykide/` 目录。
  * **Prompt 存为 MD**：生成的历史 Prompt 可保存为本地 Markdown 文件，方便回溯。

## 🛠️ 使用指南

1. **初始化**：在侧边栏点击图标打开 **YK Snidea**。
2. **连数据库**：在“数据库”选项卡配置连接，信息将保存在本地 `.ykide/db-config.json`。
3. **设规范**：在“全局设置”中定义你的原子规则，并将其组合成特定场景的“开发合集”。
4. **写需求**：在输入框通过 `@` 引用代码，用 `#` 引用表结构。
5. **拿结果**：点击“生成提示词”，复制到网页端大模型，开启高效对话。

  * **一键恢复默认**：在 Diff 和 JSON 配置面板新增“重置配置”功能，支持快速恢复系统预设提示词。
  * **文件路径显性化**：编辑器中的 `@` 标签现在显示文件的相对路径而非仅文件名，有效避免同名文件带来的混淆。
  * **检索权重优化**：改进文件搜索排序，自动降级以 `.` 或 `_` 开头的配置文件，确保核心业务代码优先展示。

### 1.0.4 (2026-03-28)

* ⚡ **多格式代码应用机制**：
  * **JSON 结构化解析支持**：新增对 AI 返回的专属 JSON 格式代码块的深度解析与替换能力。规避了复杂文本格式引发的识别异常。

### 1.0.3 (2026-03-26)

* ⚡ **核心引擎进化：代码替换与文件树**：
  * **Diff 自动化应用**：新增剪贴板解析功能。支持**网页端特殊字符清洗**（自动处理 NBSP/\u00A0）、**模糊匹配算法**（容忍手动修改导致的微小空格/缩进差异）以及 PHP 变量符号（$）的安全替换。

  * **文件树系统 (yktree)**：新增“附带文件树”选项。支持通过 `.ykide/.yktree` 配置过滤规则，并采用“智能折叠”算法防止上下文溢出。

  * **一键初始化**：新增 `init.json` 机制。插件安装后自动载入行业最佳实践的 AI 角色与 Diff 提示词配置。
  * **配置无缝迁移**：支持 Roles/Rules/Profiles 的全量导入与导出，方便在团队间同步开发规范。
  - **文件排序优化**：搜索结果现在会自动将 `.` 开头的隐藏文件或配置文件（如 .env, .gitignore）排在末尾，优先展示核心业务逻辑。

  - **架构重构**：底层消息总线升级为 `switch-case` 路由模式，将 Diff 逻辑从路由层剥离至 Service 层，符合单一职责原则。
